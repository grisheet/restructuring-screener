★ Restructuring Screener

MVP v1.0

Corporate Restructuring Intelligence

An event-driven screener for restructuring situations. Every company receives three explainable 0-100 scores — distress, restructuring significance, and turnaround opportunity — recomputed automatically whenever new events are ingested.

View on GitHub API Docs
Distress Score
0 – 100
Likelihood of financial distress
Restructuring Significance
0 – 100

API Endpoints

MethodEndpointDescription
GET/healthHealth check — returns API status and version
GET/companiesList all companies with current scores
GET/companies/{id}Full company detail with score breakdown and explainability
POST/companiesCreate a new company
GET/eventsList all ingested events
POST/eventsIngest a new restructuring event (triggers score recompute)
GET/scores/{company_id}Latest scores for a company

Quick Start

Prerequisites: Docker & Docker Compose installed.

git clone https://github.com/grisheet/restructuring-screener.git
cd restructuring-screener
cp .env.example .env        # edit DATABASE_URL if needed
docker compose up --build
# API available at http://localhost:8000
# Docs at http://localhost:8000/docs
